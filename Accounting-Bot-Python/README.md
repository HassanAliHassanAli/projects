# Accounting Bot - Balance Sheet Generator

## Overview
This is a professional Python-based accounting tool designed to automate the process of creating a **Balance Sheet**. As a Commercial Science student, I developed this tool to bridge the gap between accounting principles and data automation.

The bot collects financial data (Assets, Liabilities, and Equity), validates the accounting equation ($Assets = Liabilities + Equity$), and exports a formatted report to Excel.

## Features
- **Interactive CLI:** User-friendly prompts to enter financial data.
- **Data Validation:** Ensures all inputs are numerical and handles errors gracefully.
- **Accounting Logic:** Automatically calculates totals and checks if the balance sheet is balanced.
- **Excel Integration:** Generates a professional `.xlsx` report using `pandas`.

## Technical Stack
- **Language:** Python 3.13
- **Libraries:** Pandas, Openpyxl
- **Tools:** VS Code, Git
