# 📊 DataCo Supply Chain Analytics Dashboard

## 📝 Project Overview
This project features a comprehensive **Supply Chain Analytics Dashboard** developed in Microsoft Excel. It transforms a large-scale raw dataset (over 180,000 records) from "DataCo" into an interactive visual tool. The dashboard is designed to empower stakeholders with actionable insights into financial performance, order fulfillment, and logistical efficiency.

## 🛠️ Tools & Technologies
* **Microsoft Excel:** The core engine for data processing, analysis, and visualization.
* **Data Cleaning & ETL:** Managed a heavy dataset (~69 MB) ensuring data integrity and consistency.
* **Pivot Tables & Pivot Charts:** Used for dynamic data aggregation and multi-dimensional analysis.
* **Interactive Slicers:** Integrated for seamless data filtering by Region, Segment, and Payment Type.
* **Conditional Formatting:** Applied to the raw dataset for real-time identification of "Late Delivery Risks" and profit margins.

## 📈 Key Dashboard Features
1.  **Executive Summary:** A high-level view of core KPIs, including Total Sales, Total Profit, and Order Volume.
2.  **Product Performance Analysis:** Bar charts comparing "Sum of Sales" vs. "Sum of Benefit" across various categories (e.g., Fishing, Cleats, Cardio Equipment).
3.  **Customer Segmentation:** Analysis of sales trends across Consumer, Corporate, and Home Office segments.
4.  **Logistics Tracking:** Monitoring shipping efficiency by comparing real vs. scheduled shipping days.
5.  **Payment Method Insights:** Distribution of transactions across Cash, Debit, Payment, and Transfer methods.

## 💡 Key Business Insights
* **Scale of Operations:** Successfully analyzed **180,520 unique orders**.
* **Profitability:** Identified a total benefit of **$3,966,902.97**.
* **Top Categories:** Identified "Fishing" and "Cleats" as high-revenue drivers, while specific clothing lines showed the highest profit-to-sales ratios.
* **Risk Management:** The dashboard highlights high-risk zones for delivery delays, allowing for proactive supply chain adjustments.
