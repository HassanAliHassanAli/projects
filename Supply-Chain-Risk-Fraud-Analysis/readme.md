# 📊 Supply Chain Risk & Fraud Analysis

## 📌 Project Overview
This project provides a comprehensive analysis of supply chain data for **DataCo**, focusing on tracking financial performance, monitoring logistics efficiency, and identifying fraud risks. An interactive dashboard was built to provide decision-makers with clear insights into vital indicators to ensure business integrity and maximize profitability.

## 🎯 Key Objectives
- **Financial Analysis:** Measure and track Net Profit and total sales, identifying the most profitable product categories.
- **Geographic Distribution:** Analyze global market performance to identify high-revenue regions.
- **Logistics Efficiency:** Monitor shipping reliability and delivery statuses (Standard Class, Same Day, etc.).
- **Risk Management:** Detect suspected fraudulent orders and monitor overall order integrity.

## 🛠️ Tools & Technologies
- **Tableau:** Used for data cleaning, building interactive dashboards, and designing advanced data visualizations.
- **DataCo Smart Supply Chain Dataset:** The primary dataset used for the analysis.

## 📈 Key Insights & Dashboards
The main dashboard includes several analytical sections:
1. **Financial KPIs:** Ranking categories by Net Profit (e.g., Fishing, Cleats, Camping & Hiking).
2. **Sales Trend & Monthly Performance:** Heatmaps showing sales performance over months to identify peak periods.
3. **Global Market Distribution Map:** An interactive map showing profit distribution globally (USA leads in sales).
4. **Logistics Efficiency & Shipping Reliability:** Analysis of delivery statuses and the efficiency of different shipping modes.
5. **Order Integrity & Fraud Risk Analysis:** Precise classification of order status (Complete, Canceled, Processing) with a focus on "Suspected Fraud" cases.

## 📂 Project Structure
The repository/folder contains:
- `DataCo_SupplyChain_Analysis_Project.twb`: The core Tableau workbook containing all analyses and dashboards.
- `Dashboard.png`: A high-resolution capture of the comprehensive dashboard.
- `Global_Market_Distribution_Map.png`: A detailed view of the geographic market distribution.

## 🚀 How to Use
1. Install **Tableau Desktop**.
2. Open the file `DataCo_SupplyChain_Analysis_Project.twb`.
3. Interact with the dashboard filters (e.g., filter by Region or Order Status) to extract customized insights.
